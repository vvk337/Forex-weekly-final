# Announcements

The Announcements board allows administrators to broadcast platform-wide notifications to all employees.

## Publishing Authority
- **Publishers**: Only `OWNER`, `ADMIN`, and `SUPERVISOR` can publish.
- **Audience**: Employees receive announcements in a read-only layout feed.

## Features
- **Expiration Dates**: Optional dates when announcements expire and disappear from the active feed.
- **Pins**: Pinned announcements are pinned to the top of the feed and stand out with a distinct red brand border.
