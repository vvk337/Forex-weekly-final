# Direct Messages

Direct messaging enables private, lightweight, professional communications.

## Features
- **User Discovery**: Instantly lookup active, non-archived team members by name, department, or role.
- **Conversation Recovery**: Starting a message with a colleague automatically restores any previously deleted or archived conversation for both parties.
- **Archive Action**: Allows users to archive conversations, removing them from the main active inbox list.
- **Soft Deletion**: Deleting a conversation removes it from the calling user's inbox view while remaining fully accessible to the peer recipient.
