# 04 User Experience - Phase 8

This document highlights the frontend user experience features.

---

## 1. Bell IconDropdown
- Located in the top header control bar.
- Shows dynamic pulse badge if unread notifications exist.
- Click opens scrollable dropdown listing unread first, followed by recent read events.
- Displays action links for "Mark all as read", "Mark Read" per item, and "View Article" linking directly to publication details.
- Displays empty state: `"You're all caught up."` when empty.

---

## 2. Notification Preferences checkboxes
- Added to the bottom of the User Profile Settings screen.
- Allows employees to toggle preferences for Articles, Ticker alerts, System events, and Announcements individually.
