# 03 Editorial Navigation - Phase 5 & 6

This document details the editorial navigation filters implemented under Phase 6.

---

## 1. Filtered Horizontal Navigation tabs
- Implemented sub-navigation selector buttons for `Drafts`, `Pending Review`, `Published`, `Scheduled`, `Archived`, and `Trash` views.
- Dynamic counts are rendered in the header buttons, displaying the exact quantity of articles currently in each lifecycle stage.

---

## 2. Advanced Search & Filtering Controls
- **Search bar**: Scans for titles, authors, categories, or status codes.
- **Author**: Text matches specific authors.
- **Department**: Drops down options mapping to active departments.
- **Date Range**: Allows filtering between start and end dates.
