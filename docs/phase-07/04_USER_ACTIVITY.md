# 04 User Activity - Phase 7

This document logs User Profile activity tracking features.

---

## 1. Tracked Events
- **Logged In / Logged Out / Failed Login Attempts**: Access control trials.
- **Password Changed / Password Reset**: Profile credential changes.
- **Profile Updated**: Full name, phone, dob updates.
- **Role / Department / Workspace Changed**: Administrative modifications.
- **Archived / Restored**: Profile active states.

---

## 2. Session Metrics
Each profile console displays:
- **Last Login**: Timestamp of last successful session.
- **Current Session**: Online / Offline indicators.
- **Previous Logins**: List showing recent successful logins.
- **Failed Login Attempts**: Counter tracking unsuccessful inputs.
