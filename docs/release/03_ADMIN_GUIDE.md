# Administrator Guide

Welcome to the Forex Weekly Control Room Admin Guide. As an Administrator, you have full operational control over the system except for Owner-exclusive configurations.

## 1. User Management
- Go to the **Users** tab in the sidebar.
- Click the **Add User** button.
- Fill in Name, Username, Email, Role, Department, and Workspace.
- Choose **Send Invitation** (generates temporary credentials automatically) or **Create Temporary Password**.
- Admins can edit details, reset passwords, suspend, or archive any user (except OWNER accounts).

## 2. Managing Sponsors
- Switch to the **Marketing** Workspace.
- Navigate to the **Sponsors** tab in the sidebar.
- Edit placement headers, link URLs, or upload banner files.

## 3. Platform Monitoring
- Access the **Audit Logs** tab in the sidebar.
- Search activity logs by username, module, or date ranges to trace modifications.
