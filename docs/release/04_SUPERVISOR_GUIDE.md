# Editorial Supervisor Guide

As an Editorial Supervisor, you manage publications quality control, schedules, breaking news, and department team chats.

## 1. Editorial Workflow Approval
- Go to the **Publications** tab in the sidebar.
- Filter for `PENDING` articles.
- Select an article to view details, revise its contents, or review previous timeline comment edits.
- Click **Approve & Publish** to push live immediately, or click **Schedule** to set a future date.
- Reject articles back to `DRAFT` status with clear feedback remarks.

## 2. Breaking News Headlines
- Navigate to the **Breaking News** tab.
- Add, edit, or rearrange manual headlines overrides.
- Save settings to deploy overrides to the frontpage tickers.

## 3. Communication
- Navigate to **Inbox** -> **Groups**.
- Click **Create Group** to start workspace chat rooms.
- Navigate to **Inbox** -> **Announcements** to broadcast pinning notices.
