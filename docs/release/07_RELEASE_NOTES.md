# Release Notes - Final Release Candidate

We are excited to deliver the final refined version of the Forex Weekly Control Room.

## Key Improvements in this Release
- **Backend UI Separation**: Decoupled the backend control room completely from public website elements (TopBar headers, footer, navigation bar, widgets). Admin views are clean, fast, and secure.
- **Improved Header & Collapsible Sidebar**: Dedicated horizontal app header bar containing Weekly Control logo, Workspace selector, Global Search, Notification bell, and Profile avatar menu dropdown. Vertical sidebar is collapsible with persistent desktop settings.
- **Expanded Admin Permissions**: Re-mapped Admin privileges to grant operational management over articles lifecycle (approve, edit, trash, schedule), breaking news tickers, sponsors placement, users, and announcements.
- **Unified Search**: Removed redundant search boxes in favor of a single global search input on the header.
- **Add User Invitation Flows**: Implemented an overlay form in the users panel allowing creating profiles directly, supporting both invitation links and temporary password creation.
- **Empty States Action Items**: Re-designed empty dashboards, announcements, and chats to output clear calls to action.
